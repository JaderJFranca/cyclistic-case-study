## Dados

Os dados brutos utilizados neste projeto possuem mais de 5 milhões de registros
e mais de 1GB de tamanho, por isso não estão versionados neste repositório.

Fonte dos dados:
Cyclistic / Motivate International Inc. (dados públicos)

Para reproduzir a análise:
1. Baixe os arquivos CSV mensais no link oficial
2. Salve os arquivos no diretório:
   data/raw/
3. Execute o script em scripts/data_processing.R
